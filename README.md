<h1 align="center">Employee Management System</h1>
<h3 align="center">Full Stack CRUD Application using Spring Boot, React & Postman</h3>

---

##  Project Overview
This project is a **Full Stack Employee Management System** that performs CRUD operations (Create, Read, Update, Delete) on employee data.

It consists of:
- **Backend:** Spring Boot (REST APIs)
- **Frontend:** React JS
- **API Testing:** Postman

---

## Tech Stack
- **Backend:** Spring Boot, Java  
- **Frontend:** React JS, JavaScript  
- **Database:** (Add yours → MySQL / H2)  
- **API Testing Tool:** Postman  
- **Version Control:** Git & GitHub  

---

## ⚙️ Features
-  Add new employee  
- View employee list  
- Update employee details  
- Delete employee  
- REST API integration  
- Responsive UI using React  

---

##  API Endpoints (Sample)

| Method | Endpoint              | Description            |
|--------|---------------------|------------------------|
| GET    | /employees          | Get all employees      |
| GET    | /employees/{id}     | Get employee by ID     |
| POST   | /employees          | Add new employee       |
| PUT    | /employees/{id}     | Update employee        |
| DELETE | /employees/{id}     | Delete employee        |
